# hospital-project-SQLwithIDC
This is a repository containing files formed during sql practice by IDC
The dataset is containing 5 tables namely :
      patients
      service_weekly
      staff
      staff_schedule

patients table include data of patients such as there id,name,age,arrival_date,departure_date,service and satisfaction score towards hospital.

service_weekly table consists services provided by hospital such as service,available_beds,patients_request,patient_admitted,there refusal,satisfaction with event.

staff table consist of information about staff of the hospital,there id,name,role,service.

staff_schedule table gives the attendance information of the staffs,with columns as week,id,name,role,service,present(as 1,0)
